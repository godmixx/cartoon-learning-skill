# 🎨 卡通学习网页生成器 - AiPy Skill
## 简介
根据用户需求，自动生成小学各学科知识点的卡通风格学习网页，支持6种卡通风格、6个学科，包含知识点讲解、趣味互动练习等功能。
## 安装方式
在AiPy中导入Skill时，粘贴本仓库的GitHub URL即可。
## 使用方法
在AiPy中直接说：
```
生成 数学 三年级 乘法口诀 的卡通学习网页
生成 语文 一年级 拼音 的卡通学习网页，风格选动物乐园，难度简单
生成 英语 四年级 过去式 的卡通学习网页，风格选太空探索
```
