# 🎨 卡通学习网页生成器
> 一个AiPy Skill，根据用户需求自动生成小学各学科知识点的卡通风格学习网页。
## ✨ 功能特点
- **6种卡通风格**：卡通冒险、梦幻童话、太空探索、动物乐园、海洋世界、森林奇遇
- **6个学科**：语文、数学、英语、科学、美术、音乐
- **3个难度级别**：简单、中等、较难
- **互动练习**：每个网页包含知识点讲解和趣味互动练习
- **自动保存**：生成的HTML文件自动保存到 output/ 目录
## 📥 安装方式
在AiPy中点击「导入Skill」，粘贴本仓库的GitHub URL即可。
## 🎯 使用示例
```
生成 数学 三年级 乘法口诀 的卡通学习网页
生成 语文 一年级 拼音 的卡通学习网页，风格选动物乐园，难度简单
生成 英语 四年级 过去式 的卡通学习网页，风格选太空探索
```
## 📚 支持的知识点
### 语文
拼音、汉字、词语、句子、古诗、阅读理解、作文
### 数学
数与代数、图形与几何、统计与概率、综合实践
### 英语
字母、单词、句型、语法、对话
### 科学
生命科学、物质科学、地球与宇宙、技术与工程
### 美术
色彩、线条、形状、构图、手工
### 音乐
音符、节奏、旋律、乐器、歌曲
