# 🔊 小闯闯语音课堂生成器（纯语音版）
> 一个AiPy Skill，根据用户需求直接生成小学各学科知识点的语音讲解音频文件（MP3）。
## ✨ 功能特点
- **纯语音输出**：直接生成MP3音频文件，无需网页，无需浏览器
- **6种语音风格**：卡通冒险、梦幻童话、太空探索、动物乐园、海洋世界、森林奇遇
- **6个学科**：语文、数学、英语、科学、美术、音乐
- **3个难度级别**：简单、中等、较难
- **内置知识点库**：拼音、古诗、成语、乘法口诀、字母、植物、颜色、音符等
## 📥 安装方式
在AiPy中点击「导入Skill」，粘贴本仓库的GitHub URL即可。
## 🎯 使用示例
```
@小闯闯语音课堂生成器 生成 语文 一年级 拼音 的语音讲解
@小闯闯语音课堂生成器 生成 数学 三年级 乘法口诀 的语音讲解，风格选太空探索，难度较难
@小闯闯语音课堂生成器 生成 英语 三年级 字母 的语音讲解，风格选动物乐园
```
