# 小闯闯知识点解读生成器
## 简介
根据用户需求，生成小学各学科知识点的解读文字，支持6种风格、3个难度级别，内置丰富的知识点库。
## 使用方式
**调用格式：**
@小闯闯知识点解读生成器 生成 [学科] [年级] [知识点] 的知识点解读，风格选[风格]，难度[简单/中等/较难]
**参数说明：**
| 参数 | 说明 | 可选值 |
|------|------|--------|
| 学科 | 小学学科 | 语文、数学、英语、科学、美术、音乐 |
| 年级 | 年级 | 一年级~六年级 |
| 知识点 | 具体知识点 | 拼音、古诗-静夜思、成语-画蛇添足、乘法口诀、字母、植物、颜色、音符等 |
| 风格 | 文字风格（可选，默认卡通冒险） | 卡通冒险、梦幻童话、太空探索、动物乐园、海洋世界、森林奇遇 |
| 难度 | 难度级别（可选，默认中等） | 简单、中等、较难 |
**调用示例：**
@小闯闯知识点解读生成器 生成 语文 一年级 拼音 的知识点解读
@小闯闯知识点解读生成器 生成 数学 三年级 乘法口诀 的知识点解读，风格选太空探索，难度较难
## 执行流程
1. 解析用户输入，提取学科、年级、知识点、风格、难度
2. 从内置知识点库中查找对应的解读内容
3. 根据风格添加开头和结尾的趣味引导语
4. 返回完整的知识点解读文字
